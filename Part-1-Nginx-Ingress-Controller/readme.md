# Getting started with use cases

1. Clone this repository

```code
git clone https://github.com/tareqaminul/f5-nginx-redhat-ocp-ph26.git
```

2. Change directory

```code
cd f5-nginx-redhat-ocp-ph26/Part-2-Nginx-Gateway-Fabric
```

## Running use cases

- [Lab 1](1.basic) - Basic URI-based routing
- [Lab 2](2.advanced-routing) - Advanced routing using HTTP matching conditions
- [Lab 3](3.access-control) - IP-based Access Control
- [Lab 4](4.cross-namespace-configuration) - Cross Namespace Routing
- [Lab 5](5.jwt) - JSON Web Token - JWT-based Authentication
- [Lab 6](6.rate-limit) - Rate Limiting
- [Lab 7](7.tls) - TLS Offloading
- [Lab 8](8.traffic-splitting) - Traffic Splitting
- [Lab 9](9.waf) - F5 WAF for NGINX, Natively In OpenShift
